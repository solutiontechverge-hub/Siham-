# Fullstack Setup (Node backend + Next frontend)

## Structure
- `backend` => Node.js + Express + PostgreSQL
- `frontend` => Next.js + MUI

## Prerequisites
- Node.js 18+ and npm
- PostgreSQL running locally

## Run Backend
```bash
cd backend
cp .env.example .env
npm install
npm run dev
```

## Run Frontend
```bash
cd frontend
npm install
npm run dev
```

Then open `http://localhost:3000`.
